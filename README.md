# poetry-generator

Random poetry line generator featuring confessional poets

## About

This is a web-based poetry line generator that displays curated quotes from contemporary poets. Users can generate random lines and save their favorites to a collection list.

## Features

- Random poetry line selection from curated collection
- Display of poem title and author attribution
- Save favorite lines to a persistent list
- Clean, readable typography optimized for poetry

## What I Learned

- DOM manipulation (getElementById, innerHTML, createElement)
- Event handlers (onclick functions)
- Working with arrays of objects
- Math.random() for random selection
- Styling text with multiple font weights and styles
- Managing current state in JavaScript

## How to Use

1. Open `index.html` in a browser
2. Click "New Inspiration" to see a random poetry line
3. Click "Save to Favorites" to add the current line to your collection
4. View all saved lines in the "Collected Lines" section below
